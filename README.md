## How to Install and Run
安裝

1. 從 Github 上 clone 這個專案：
`git clone https://github.com/{YOUR_USERNAME}/{REPO_NAME}.git`
2. `npm install`

運行

1. 運行前端應用：
`npm start`
2. 運行後端應用：
`npm run dev`

## Technology Stack
前端：React、CSS、JavaScript
